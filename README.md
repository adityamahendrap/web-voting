# Web Musma Teknik 2023
Origin: https://github.com/komenkx1/poling

``` bash
# install dependencies
$ composer update
$ composer install
$ npm install

# env
$ cp .env.example .env

# generate key
$ php artisan key:generate

# run migration
$ php artisan migrate

# run seeder
$ php artisan db:seed

# run dev server
$ php artisan serve
```
